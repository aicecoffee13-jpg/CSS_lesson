## ボックスモデル

* ブロックボックス
上下に配置される pタグ、divタグなど
幅、高さの指定が通る
横に余白があった場合は、改行される。


* インラインボックス
自分の中に入っているコンテンツ量しか表示されない
aタグ、spanタグなど

* インラインブロックボックス
インラインとブロックの特性を併せ持つ

## 単位
* em = 1文字 = 1em
場所によって変化するので注意
フォントサイズの設定よりも余白などに使用すると便利
* vh/vw
ブラウザの画面サイズに応じて変化する。

* clamp
調べる

* 特殊文字
使用する際は、コードを調べる。

##　要素の外側の余白
* margin余白を付ける位置(「,」はスペース)
margin: 上, 左, 下, 右；
margin: 上, 左右, 下；
margin: 上下, 左右；　
margin: 上下左右；　

* コンテンツを真ん中に寄せたいとき
margin: 0 auto;と記述する
ブロックボックスだけ適用されるが、インラインボックスは適用されない。
幅の指定は必ず行う。

* マージンの重なり
大きい方が採用される。

## 内側の余白
* padding
余白を付ける位置はmarginと一緒
外側に膨らむ。

##　線
* border
borderはpxで記述する。
borderもmarginと一緒の位置（指定の位置）

* borderの注意点
borderは外側に膨らむ。

* box-sizing:border-box
box-sizing: border-box;は内側に膨らむ
border,paddingで使用

## calc
* 計算機：単位が異なっていても計算できる

## メモ
* htmlのフォントサイズ基準は、16pxである。html{font-size:100%}は16pxで設定されている。1remが16pxで設定されている。2remは32pxである。ブラウザの設定で変えた際にremは％に対して大きさが変化する。
html{font-size:62.5%}は10pxで設定される。その為、20pxにしたい際には2remにする。
